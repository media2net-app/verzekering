## Installatie

```bash
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in je browser.

## Build

```bash
npm run build
npm start
```

## Technologie

- Next.js 14
- React 18
- TypeScript
- CSS Modules
